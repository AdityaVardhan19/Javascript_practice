Functions & Scope

Functions are reusable blocks of code that perform a specific task. Scope determines where variables can be accessed in a program.

1. Functions

A function is a reusable block of code used to perform a particular task. 

function functionName(){

}

2. Parameters

Parameters are values received by a function. 

function add(a, b){

}

3. Return Keyword

The return keyword sends a value back from a function. It allows the returned value to be stored, reused, or displayed later. 

function add(a, b){

    return a + b;

}

4. Arrow Functions

Arrow Functions provide a shorter and modern way to write functions. 

const greet = () => {

    console.log("Hello");

}

5. Function Call

add(10, 20);

6. Scope

  • Global Scope → Accessible throughout the program.
  • Local Scope → Accessible only inside a function.
  • Block Scope → Variables declared using let and const are accessible only inside the block { }.
