Operators & Conditional Statements

Operators are used to perform calculations, comparisons, and logical operations on variables. Conditional statements use these operators to make decisions based on different conditions.

JavaScript Operators

1. Arithmetic Operators

Used to perform mathematical calculations. 

+    Addition
-    Subtraction
*    Multiplication
/    Division
%    Modulus
**   Exponent

2. Assignment Operators

Used to assign or update values of variables. 

=
+=
-=
*=
/=
%=

3. Comparison Operators

Used to compare two values. The result is either true or false. 

>
<
>=
<=
==
===
!=
!==

4. Logical Operators

Used to combine multiple conditions. 

&&   AND
||   OR
!    NOT

5. Increment & Decrement Operators

Used to increase or decrease a variable value by one. 

++
--

6. Ternary Operator

A shorthand way to write an if...else statement. 

condition ? trueValue : falseValue;

7. Typeof Operator

Used to identify the data type of a variable. 

typeof variableName;

Conditional Statements

1. if Statement

Executes a block of code only if the specified condition is true. 

if(condition){

    // code

}

2. if...else Statement

Executes one block if the condition is true, otherwise executes another block. 

if(condition){

    // code

}
else{

    // code

}

3. if...else if...else Statement

Used to check multiple conditions one after another. 

if(condition){

    // code

}
else if(condition){

    // code

}
else{

    // code

}
