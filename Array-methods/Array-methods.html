JavaScript Array Methods

Array methods make working with arrays easier. Instead of writing loops repeatedly, JavaScript provides built-in methods to transform, filter and calculate data.

2. map()

map() creates a new array by modifying every element. 


array.map((item)=>{

return item;

})

3. filter()

filter() returns only the elements that satisfy a condition. 


array.filter((item)=>{

return condition;

})

4. reduce()

reduce() combines all array elements into one final value. 


array.reduce((total,item)=>{

return total+item;

},0)

5. Difference

Method Purpose Returns
map() Modify New Array
filter() Select New Array
reduce() Calculate Single Value
