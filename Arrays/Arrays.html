Introduction to Arrays

An array is a special variable that stores multiple values in a single variable. Each value is stored at a specific index.

1. What is an Array?

An array stores multiple values inside a single variable. 

let fruits = ["Apple","Mango","Orange"];

2. Why Arrays?

Instead of creating many variables, we can store all values in one array. 

let student1 = "Rahul";
let student2 = "Ravi";

↓

let students = ["Rahul","Ravi"];

3. Creating Arrays

let fruits = ["Apple","Mango","Orange"];

4. Array Index

Array indexing starts from 0. 

Index : 0      1        2

        Apple  Mango  Orange

5. Accessing Elements

fruits[0]

fruits[2]

6. Array Length

fruits.length

7. push() and pop()

push() → Adds an element at the end. 

fruits.push("Banana");

pop() → Removes the last element. 

fruits.pop();

8. shift() and unshift()

unshift() → Adds an element at the beginning. 

fruits.unshift("Apple");

shift() → Removes the first element. 

fruits.shift();

9. splice()

Used to add, remove or replace elements. 

array.splice(start,deleteCount,item);
Remove 
fruits.splice(1,1);
Add 
fruits.splice(1,0,"Banana");
Replace 
fruits.splice(1,1,"Grapes");

10. slice()

Creates a copy of a portion of an array without modifying the original array. 

let result = fruits.slice(1,3);

11. splice() vs slice()

splice() slice()
Modifies Original Array Does Not Modify Original Array
Add / Remove / Replace Copies Elements
